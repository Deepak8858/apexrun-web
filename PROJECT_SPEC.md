1\. Project Vision

ApexRun is a high-performance running platform that bridges the gap between Strava's social engine and Runna's AI coaching. This website serves as a technical showcase for my professional resume, emphasizing complex system architecture (Flutter + Go + PostGIS) and agentic AI integration (Gemini 1.5 Flash).
